  1. !DOCTYPE : All Html files should start with this !DOCTYPE and basic Syntax of the html files 
                           
<!DOCTYPE html>
<html>
 <head>
   <title>
     TitleName
   </title>
 </head> 
 <body>
 In body we can represent any tags
 </body>
</html>
  2. <a> tag : The <a> tag defines a hyperlink, which is used to link from one page to another. 
                           
<a href="https://www.akhilchapa.com">Visit akhilchapa.com!
    Attributes that can be linked in <a> tag

    • download:Here the location is to download the item from it. 
                                     
<a href="/images/image.jpg"download >
    • Target attribute: Used for making the clicked link to show according to the specified 
                                  
<a target="_blank|_self|_parent|_top|framename">
<a href="https://www.akhilchapas.com" target="_blank">Visit akhl.com</a>
    • rel attribute: 
                                  
<a href="https://www.akhilchapa.com" type="text/html">akhils</a>
    • How to link to mail someone: 
                                  
<a href="mailto:someone@example.com">Send email</a>
    • How to link to a phone number: 
                                  
<a href="tel:+4733378901">+47 333 78 901</a>
    • How to link to a JavaScript: 
                                  
<a href="javascript:alert('Hello World!');">Execute JavaScript</a>
